These waifu-*.png files are the pinned WezTerm backgrounds shipped with this repo.
The bootstrap scripts copy ALL of them to ~/.config/wezterm/backgrounds/, and
wezterm.lua picks one at RANDOM each time it loads (every new WezTerm launch).

- Add more images here (any *.png/*.jpg/*.jpeg/*.webp) to expand the rotation.
- If the folder is empty, the scripts fall back to downloading 7 random SFW
  anime images from nekos.best.
